# 👻 Phantom Hacks
**India's Hackathon Discovery Engine**

## Overview
Phantom Hacks is a dedicated static website allowing users to discover and register for hackathons across India. 

## Pages
- `index.html` — Main discovery page with dynamic lists.
- `profile.html` — User profile, tracking, and saved hackathons.
- `post.html` — Post or submit a new hackathon.

## Tech Stack
- **Frontend:** Vanilla HTML5, CSS3, JavaScript
- **Backend & Auth:** Supabase
- **Hosting:** Vercel

## Run Locally
1. Clone the repository.
2. Open the project folder in VS Code.
3. Use the **Live Server** extension to host on `http://localhost:5500`.

## Database Config
Managed directly via Supabase MCP connector — no SQL migration files needed. The project is fully API-driven from the browser context on the client side.
